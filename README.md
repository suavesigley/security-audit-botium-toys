# Security Audit — Botium Toys

## Overview
This project contains a security audit conducted for Botium Toys to evaluate security controls and compliance posture.

## Objectives
- Identify missing controls
- Evaluate compliance gaps
- Assess organizational risk
- Provide remediation recommendations

## Controls Assessment
The following controls were evaluated:

- Least privilege
- Disaster recovery
- Password policies
- Separation of duties
- Firewall
- IDS
- Backups
- Encryption
- Physical security
